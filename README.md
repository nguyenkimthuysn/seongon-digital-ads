# SEONGON Digital Ads – Landing Page

Landing page giới thiệu dịch vụ Digital Ads của SEONGON Agency.

## Live Demo

**https://nguyenkimthuysn.github.io/seongon-digital-ads/**

## Tính năng

- Giới thiệu hệ sinh thái Digital Ads toàn diện (Google Ads, Facebook Ads, TikTok Ads, Zalo Ads, Landing Page, CRM)
- Form kiểm tra sức khỏe tài khoản quảng cáo miễn phí (gửi dữ liệu về Google Sheets)
- Form liên hệ tư vấn (gửi dữ liệu về Google Sheets)
- Bảo mật thông tin CID tài khoản quảng cáo
- Responsive, single-file HTML (không dùng thư viện JS ngoài)

## Công nghệ

- HTML / CSS / JavaScript thuần
- Google Apps Script Web App (backend lưu form vào Google Sheets)
- GitHub Pages (hosting)

## Cấu trúc file

| File | Mô tả |
|------|-------|
| `index.html` | Landing page chính |
| `google-apps-script.js` | Code Google Apps Script deploy lên Google Sheets |
